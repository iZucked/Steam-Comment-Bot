Data Perspective: Editing Views

Cooldown Costs

  1. Links Load ports to a price curve to use when pricing cooldowns at that load port.

  2. Multiple load ports can be grouped together, but a load port may only have a single price curve.

Ports

  1. Lists all the available ports in the scenario

  2. Manage distances and routes via toolbar buttons

Vessels

  1. Lists all the known vessel and classes in the scenario

  2. Manage base fuels via toolbar button

  3. Note: Use editor fleet tab to register a vessel as a fleet vessel.

Canal costs

  1. One row per vessel class and non direct route.

  2. Automatically created/removed

  3. Specifies the fixed costs for using a canal.

  4. Note: Consumptions are edited from the vessel class editor.

Port Costs

  1. Links operations costs for a given port.

  2. Multiple ports can be grouped together, but a port may only have a single single set of costs.

  3. Costs are fixed unless a reference capacity is provided. If a reference capacity is provided then port costs are scaled linearly in proportion to the actual vessel capacity.

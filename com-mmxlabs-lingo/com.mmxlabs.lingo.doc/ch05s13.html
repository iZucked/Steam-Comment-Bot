Moving an installation of LiNGO

  1. ZIP up the LiNGO application folder and copy to the new location on the new computer.

  2. Re-create any shortcuts.

  3. Launch LiNGO and check everything works!

  4. Note, the workspace (which holds the scenario data) will be in a folder called "LiNGO" in your user home directory. We assume that is copied over automatically, but if not then you'll need to also copy that folder from the home directory on the old computer to that on the new computer.

  5. <Check LiNGO/mmxlabs directories in user home - e.g. lingo.pks>

You can of course just copy across a base case and start afresh, if you're absolutely certain you don't need any of the old data.

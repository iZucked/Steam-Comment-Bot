Commercial Tab

The commercial tab shows the portfolio’s purchase and sales contracts and corporate entities which own contracts or vessels.

The Purchase Contracts and Sales Contracts grids allow you to manage your purchase contracts and sales contracts respectively. Each entry edited in more detail by double clicking or hitting <Return> to bring up the Contract Editor.

The Contract Editor dialog allows the editing of a number of fields relating to a purchase or sales contract. These fields are organised in three tabs: “Contract”, “Pricing” and “Restrictions”.

Contract Tab

The “Contract” tab contains various fields to edit the basic attributes of the contract.

Name

A unique name to denote the contract.

Entity

The legal entity owning the contract.

Ports

A list of allowed purchase/sale ports.

Preferred Port

A default port for the contract (editing convenience only, not optimisation).

Min Quantity

The minimum volume for a transaction under this contract, in m3.

Max Quantity

The minimum volume for a transaction under this contract, in m3.

Notes

Any notes regarding the contract.

Contract Type

The types of trade (FOB or DES) permitted under this contract.

Cancellation Fee

An amount in dollars payable if a slot using the contract is not used.

Cargo CV

(For purchase contracts only) the calorific value of the LNG purchased.

Pricing Tab

The “Pricing” tab allows editing of parameters which determine pricing. Complex contracts will have custom parameters which can be edited. For the simple contracts, this is just a price expression (a formula using standard price indices (see “Price Expressions”).

Restrictions Tab

The “Restrictions” tab specifies of any restrictions applying to a transaction conducted under this contract.

Restrictions Are Permissive

If this checkbox is unchecked, the “Contracts” and “Ports” inputs specify lists of contracts and ports which the complementary transaction may not be associated with. If the checkbox is ticked, the complementary transaction must be associated with a contract / port in the specified list. In either case, a blank list is treated as no restriction.

Contracts

A list of contracts which the complementary transaction (i.e. a sale, for a purchase contract, or a purchase, for a sales contract) must (or must not) be conducted under.

Ports

A list of ports which the complementary transaction (i.e. a sale, for a purchase contract, or a purchase, for a sales contract) must (or must not) be conducted at.

Min / Max CV Value

(Sales contracts only) These fields allow you to specify restrictions on the calorific value of the LNG sold under the contract.

Sales Delivery Type / Cargo Type

“Shipped” if the trade must involve shipping, “Not Shipped” if the trade must be a FOB or DES only transaction, “Any” if no restrictions are involved.

Entity Editor

The Entity Editor allows you to modify a variety of fields associated with a legal trading entity. The Shipping Book and Trading Book panels allow tax rates to be set, by commencement dates.

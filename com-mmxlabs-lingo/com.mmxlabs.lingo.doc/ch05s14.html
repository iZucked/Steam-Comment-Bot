Automatic Scenario Data Backup

Your scenario data can be automatically backed up on application startup and shutdown. This creates a zip file of all your scenario data and stores it in a destination folder of your choice, overwriting any previous backup. This can be a local folder, or a mapped remote folder.

To enable the backup, locate the "Scenario Data Backup" preference page (under the Window Menu, Preferences option). Here you can specify the name of the archive and the destination folder (this must exist). Tick the enabled check box to enable the backup. Restart the application to trigger the backup process.

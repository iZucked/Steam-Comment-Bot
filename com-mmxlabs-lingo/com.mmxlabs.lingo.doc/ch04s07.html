Contract Fields

Name

Unique name of the contract

Entity

Owning corporate entity

Ports

The ports linked to the contract (i.e. load port for purchase contracts, discharge ports for sales contracts). Can be left blank

Preferred Port

A preferred, or default, port for the contract (used when creating new slots).

Min/Max Quantity

Default slot minimum and maximum transfer volume in m3.

Notes

Free text field to user notes on the contract.

Contract Type

Whether this contract is just for FOB or DES transactions or both. For purchase contracts this means FOB Purchase or DES Purchase. For sales contracts this means DES Sales or FOB Sales.

Cancellation Fee

Fixed fee for not using a slot linked to this contract.

Cargo CV

(Purchase Only) Contract purchase CV if different to CV defined on the load port.

The Restrictions Tab

Restrictions Are Permissive

Flips restrictions from blacklisted contracts and ports to whitelisted contracts and ports.

Contracts

List of contracts which cannot be linked to this contract (or if restrictions are permissive then this is the list of contracts which can only be linked to this one).

Ports

List of ports which cannot be linked to this contract (or if restrictions are permissive then this is the list of ports which can only be linked to this one).

Min/Max CV Value

(Sales Only) Only link to purchases with a CV value within this range.

Cargo Type

(Sales Only) Shipped means only link to FOB Purchases, Non Shipped means only link to DES Purchases.

Sales Delivery Type

(Purchase Only) Shipped means only link to DES Sales, Non Shipped means only link to FOB Sales.
